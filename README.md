# project-management-demo-frontend
Project management tool demo app.

## Set up
```
yarn install
yarn codegen
```

## Start server

```
yarn dev
```

## Test

Generate query and mutation mock.
```
yarn codegen
yarn new:queryMock
yarn new:mutationMock
```

Run tests.
```
yarn test:unit
yarn test:integration
```

## Build
```
yarn build
```
